## 什么是鼠标手势软件？WGestures 2 解决什么问题

**鼠标手势软件**，简单说就是：你用鼠标划出一个特定轨迹（比如向上划、向左下划、画字母 L），软件识别这个手势，然后执行你预设的命令。

WGestures 2 把这件事做到了"全局"层面——不只是浏览器里能用，而是你电脑上几乎所有软件都能响应这些手势。文件管理器、PS、Word、视频播放器，统统通吃。

**它解决的核心问题：**减少你在鼠标操作和键盘快捷键之间来回切换的次数。那些你记不住快捷键、或者懒得去点菜单的操作，全部可以绑到手势上。

## WGestures 2 有哪些功能？核心亮点一览

### 全局触发，不挑软件

按住鼠标右键（或中键、侧键）画手势，在几乎所有应用程序里都能用。这是它比浏览器手势插件强的地方——你不需要给每个软件单独装扩展。

### 屏幕四角触发

把鼠标指针滑到屏幕的某个角落，就能触发设定好的动作。Mac 用户对这个功能很熟悉，WGestures 2 把它带到了 Windows 上，也让 Mac 版有了更多配置灵活性。

### 修饰键叠加，一个手势变多个

同样是「向右划」这个手势：不按修饰键是「前进」，按住 Ctrl 再划可以设成「切换到下一个标签页」，按住 Alt 划又是另一个命令。用这个方法，少量手势能覆盖大量操作，减少记忆负担。

### 手势录制与可视化轨迹

创建手势时可以录制自己的划动轨迹，还能回放动画、搜索手势、拖拽复制。设置界面做得比较直观，不需要摸索太久。

### 脚本与程序支持

支持运行指定程序、执行按键序列、窗口控制操作，以及 AppleScript、Lua、Cmd 等脚本。有一定技术基础的用户可以把这个工具的可能性拓展得很深。

### 自定义菜单（仅 Windows）

可以把一组相关操作放进一个菜单，用一个手势呼出。比如「文件操作菜单」里放移动、重命名、复制路径等命令，一次手势搞定多件事。这个功能目前是 Windows 专属，Mac 版暂不支持。
